# How to start

```
git clone https://github.com/RuixuanLi97/translate_test.git
npm install
npm install -g typescript (If you do not have this)
npm install -g ts-node (If you do not have this)
npx ts-node translate_i18n.ts
```

# Usage

```
npx ts-node translate_i18n.ts
npx ts-node delete_key.ts fr.ts "RECEPTION.MESSAGE.TEST"
```
